# Waitlist 页面文案（可直接复制到网站）

标题：
- Changelog-to-Impact Copilot

副标题：
- Stripe / OpenAI 更新后，自动告诉你哪些代码会受影响，并生成升级 Issue。

适用人群：
- 依赖 Stripe + OpenAI 的小团队 SaaS（TS/Node 优先）

你会得到：
- 变更摘要（breaking / deprecation）
- 受影响文件/行号列表
- 升级行动清单（可直接创建 GitHub Issue）

为什么做这个：
- 读 changelog 很容易，难的是知道“你的代码哪里会坏”。

加入内测：
- 留下邮箱，我们会邀请你进入托管版 beta（Slack 通知、多 repo、团队协作）。

FAQ：
Q：你们会读取我的私有代码吗？
A：CLI 版本完全在本地跑。托管版会提供最小权限与可选自托管。

Q：支持哪些语言？
A：v0.1 优先支持 TS/JS。后续会扩展。

Q：会自动开 PR 吗？
A：第一版先生成报告与 Issue，后续支持 PR。
